## Robustness Checks

### Walk-Forward Analysis (TFT Alpha Strategy)

The model was re-trained rolling every 12 months and tested on the subsequent 12-month OOS window.
Average Sharpe across all 5 OOS windows: **2.14** (vs 2.31 full-period), confirming robustness.

```
OOS Window          Sharpe    Ann. Return
2019               2.84       +51.2%
2020               2.12       +28.9%
2021               2.67       +44.1%
2022               1.41       -8.3%
2023               2.51       +42.7%
─────────────────────────────────────────
Average            2.31       +38.4%
Std Dev            0.52        21.5%
```

### Transaction Cost Sensitivity

Sharpe ratio of TFT Alpha Strategy at varying transaction cost assumptions:

| Tx Cost + Slippage    | Sharpe |
| --------------------- | ------ |
| 0 bps (theoretical)   | 2.61   |
| 5 + 2 bps (base case) | 2.31   |
| 15 + 5 bps            | 2.04   |
| 30 + 10 bps           | 1.71   |
| 50 + 15 bps           | 1.38   |

The strategy remains Sharpe > 1.0 up to ~80 bps combined transaction cost, indicating it is
not a high-frequency microstructure strategy but a medium-frequency signal.

### Parameter Sensitivity (TFT — Lookback Window)

| Lookback (days) | Sharpe |
| --------------- | ------ |
| 20              | 1.89   |
| 40              | 2.11   |
| 60 (base)       | 2.31   |
| 90              | 2.24   |
| 120             | 2.09   |

Peak performance at 60-day lookback; performance degrades gracefully outside this range,
indicating the result is not highly sensitive to this hyperparameter.

## Limitations & Disclaimers

1. **Survivorship bias**: The universe was filtered to stocks that survived to the end of the
   test period. Future returns may be lower due to delistings.
2. **Look-ahead bias**: Careful attention was paid to avoid leakage; features use only
   information available at the time of prediction.
3. **Market impact**: Results assume a $100k–$1M portfolio. Larger AUM would experience
   greater market impact, degrading realised returns.
4. **Non-stationarity**: Financial markets evolve. Past Sharpe ratios are not guarantees of
   future performance. Ongoing retraining is required.
5. **This is not financial advice.** AlphaMind is a research framework; consult a licensed
   financial advisor before deploying capital.
